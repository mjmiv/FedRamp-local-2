## Project Intake Copilot
Prompt: Classify this project request, detect duplicates, suggest a project type, and identify missing intake fields.

Prompt: Match this project to eligible funding sources, explain restrictions, and recommend an allocation strategy.

Prompt: Summarize the selected scenario for an executive audience using financial, risk, and delivery impacts.

## Asset Intelligence Copilot
Prompt: Analyze asset condition, lifecycle risk, remaining useful life, and recommend capital or maintenance interventions.

## Operations Copilot
Prompt: Prioritize open work orders, suggest crew assignments, and identify backlog reduction opportunities.
