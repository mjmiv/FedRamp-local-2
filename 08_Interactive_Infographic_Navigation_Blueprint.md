# MJM-Graphics v4 Interactive Infographic Navigation Blueprint

## Core Pattern
Overview -> Domain -> Capability -> Feature -> Insight Panel -> CTA.

## Domain Click
Opens a capability ring or panel for the selected Level 1 process domain.

## Capability Click
Expands the capability into features, KPIs, personas, and AI opportunities.

## Feature Click
Opens a detail panel containing:
- Business value
- Primary screen
- KPI
- Persona
- Marketing narrative
- Demo narrative
- AI service
- Related data objects

## CTA Types
- Request demo
- Explore workflow
- View sample dashboard
- Compare scenarios
- Generate executive summary

## Phase 1 Build Recommendation
Start with these high-impact clickable paths:
1. Candidate Intake Management
2. Funding Optimization
3. Scenario Development
4. Executive Review
5. Schedule Management
6. Condition Modeling
7. Dispatch Management
